Problem

The Challenge

With the rapid increase in digital financial transactions, individuals often find it difficult to monitor, control, and optimize their spending effectively. Existing personal finance tracking systems mainly provide transaction recording and basic visualization features but lack intelligent mechanisms to analyze historical spending patterns and predict future budget requirements.

As a result, budgeting remains manual, non-personalized, and reactive, increasing the risk of overspending.

Therefore, there is a need for a data-driven and intelligent financial management system that integrates machine learning techniques — specifically XGBoost regression — to provide optimized and personalized budget recommendations.

Literature Review / Market Research

  • • Existing apps like Mint and Walnut focus on transaction recording and categorization only
  • • Dashboards and graphical visualization are the primary user-facing features
  • • ML models, especially Gradient Boosting (XGBoost), perform effectively in regression-based forecasting
  • • XGBoost handles structured datasets and captures complex non-linear spending trends
  • • Most current finance systems do not integrate advanced predictive models for budget recommendations

Research Gap / Innovation

  • • Most personal finance tools focus only on tracking and visualization
  • • Budget planning remains manual, reactive, and user-dependent
  • • Limited integration of machine learning in personal finance apps
  • • No analysis of historical patterns to predict future budget needs
  • • This project integrates XGBoost regression to generate intelligent, personalized budget recommendations
  • • Transforms a static tracker into a dynamic, data-driven financial decision-support system
Methodology

System Methodology

Dataset / Input

  • • 26,052 transaction records
  • • 6 key features per record
  • • Attributes: City, Date, Card Type, Expense Type, Gender, Amount
  • • 986 cities · 4 card types · 6 categories
  • • No missing values — clean, consistent data
  • • Preprocessing: encoding, feature selection, pivot aggregation

Model / Architecture

  • • Algorithm: XGBoost Regressor
  • • Predicts spend ratio (next ÷ current month)
  • • Handles non-linear financial patterns
  • • L1 & L2 regularization prevents overfitting
  • • Safety clamp: 0.8× – 1.5× of current spend

Conclusion

This project successfully delivers an AI-powered personal finance tracker integrating XGBoost regression for budget prediction, interactive Plotly visualizations, Claude AI chatbot advisory, and a full CRUD transaction system with CSV export — all within a professional Streamlit web application. The system transforms reactive expense recording into a proactive, data-driven financial management experience, achieving an R² score of 0.91 with realistic user-anchored predictions.
