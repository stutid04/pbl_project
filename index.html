AI-Based Personal Finance Tracker and Budget Recommender

"Project Statement: "The primary objective of this project is to transform a traditional transaction-based finance tracker into an intelligent, data-driven system that provides optimized budget recommendations using machine learning."

Problem Statement

With the rapid increase in digital financial transactions, individuals often find it difficult to monitor, control, and optimize their spending effectively. Existing personal finance tracking systems mainly provide transaction recording and basic visualization features but lack intelligent mechanisms to analyze historical spending patterns and predict future budget requirements.As a result, budgeting remains manual, non-personalized, and reactive, increasing the risk of overspending. Therefore, there is a need for a data-driven and intelligent financial management system that integrates machine learning techniques to provide optimized and personalized budget recommendations.

Literature Review / Market Research

  • • Existing personal finance applications such as Mint and Walnut focus on transaction recording and expense categorization.
  • • These systems provide dashboards and graphical visualization of spending patterns.
  • • They primarily help users monitor financial activities through summary reports.
  • • Research in financial analytics highlights that Machine Learning models, especially Gradient Boosting algorithms like XGBoost, perform effectively in regression-based forecasting.
  • • XGBoost can handle structured datasets and capture complex, non-linear spending trends.
  • • However, most current finance systems do not integrate advanced predictive models for personalized budget recommendations.

Research Gap / Innovation

  • • Most existing personal finance systems focus only on tracking and visualization.
  • • Budget planning remains manual, reactive, and user-dependent.
  • • There is limited integration of machine learning models in personal finance applications.
  • • Current systems do not analyze historical spending patterns to predict future budget needs.
  • • This project integrates an XGBoost regression model to generate intelligent and personalized budget recommendations.
  • • The approach transforms a static tracker into a dynamic, data-driven financial decision-support system.

System Methodology

Dataset / Input

  • • Structured financial dataset with 26,053 transaction records
  • • 6 key features including categorical and numerical attributes
  • • Attributes: City, Date, Card Type, Expense Type, Gender, Amount
  • • No missing values ensuring data consistency
  • • Preprocessing performed using Pandas & NumPy
  • • Preprocessing: Data cleaning, encoding categorical variables, feature selection
  • • Dataset used for training and evaluating XGBoost regression model

Model / Architecture

  • • Algorithm Used: XGBoost Regressor (Gradient Boosting Framework)
  • • Sequential tree-based learning to minimize residual errors
  • • Handles non-linear financial patterns effectively
  • • Includes L1 & L2 regularization to prevent overfitting
  • • Performance evaluated using R² Score and error metrics
